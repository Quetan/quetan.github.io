About technology

Mesh networks

We offer simple solutions for creating Wi-Fi networks. These solutions are based on technology of peer-to-peer (mesh) networks. We offer the tools to build new Internet infrastructure, excluding last mile providers as access intermediaries. The "last mile" infrastructure can be created by the users themselves, using our software and non-professional equipment. Thanks to the user infrastructure, it will be possible to reduce the cost of the Internet and organize access in small settlements, and increase the network capacity in cities. Using our software solutions and your equipment, we together build a global Wi-Fi network with a single authorization.

Technology advantages

  1. Ease of installation
  2. Low cost of Wi-Fi coverage
  3. Easy management of the created network segment through the portal
  4. The Ability to work in the Ternex global Wi-Fi network or create your own private segments
  5. The ability to display advertisements both in your own segment of the network and in the entire Ternex network

How it works?

Step 1.

You connect to the Internet a router with software developed by us. This can be wired, mobile or Wi-Fi access.

Step 2.

The router will automatically create a Ternex Wi-Fi network to which you and other users can automatically connect. It works straight away, out of the box, and does not require additional payment.

Step 3.

Authorization on the network goes through a single account. The confirmation of which is carried out by phone number and complies with the requirements of the legislation on public Wi-Fi.

Step 4.

By registering once, the user gets access to all network segments without geo-referencing.

Step 5.

The user uses the free Internet, but views ads when connected.

Step 6.

To increase network coverage, simply place new routers within range of the first or subsequent router. No wiring required.

Step 7.

You can create new, additional networks with personal authorization.

Step 8.

Router owners receive a portion of advertising revenue and a portion of cryptocurrency emissions.
